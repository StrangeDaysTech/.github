# CLAUDE.md

Guía para Claude Code al trabajar en este repositorio.

## Qué es este repositorio

Es el **repositorio especial de perfil de organización de GitHub**
(`StrangeDaysTech/.github`) de **Strange Days Tech, S.A.S.**, una empresa
mexicana dedicada al diseño, desarrollo e implementación de soluciones
tecnológicas (software a medida, herramientas para programadores, productos
SaaS e inteligencia artificial).

No contiene código de aplicación: solo archivos Markdown. No hay build,
dependencias ni tests.

## Cómo funciona el perfil

GitHub renderiza automáticamente `profile/README.md` en la página pública de la
organización (`github.com/StrangeDaysTech`). **Para cambiar lo que ven los
visitantes del perfil, edita `profile/README.md`.**

## Estructura

| Archivo | Propósito |
|---------|-----------|
| `README.md` | README del propio repositorio (explica qué es). No se muestra en el perfil. |
| `profile/README.md` | Contenido público que GitHub muestra en la página de perfil de la organización. |

Nota sobre rutas: el repositorio git vive en el subdirectorio `.github/`. El
directorio de trabajo padre (`github-profile/`) no es un repositorio git.

## Convenciones

- Todo el contenido se escribe en **español**, con ortografía y acentos
  correctos (á, é, í, ó, ú, ñ).
- Es Markdown puro: no hay pasos de compilación, dependencias ni pruebas que
  ejecutar.
- Cambios al perfil público = editar `profile/README.md`.

## Git

- Rama principal: `main`. Remoto: `origin`
  (`https://github.com/StrangeDaysTech/.github.git`).
- Haz commit o push solo cuando el usuario lo pida explícitamente.
